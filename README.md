<h5 align=center>Easy access to mock test data powered by <a href="https://chancejs.com">chance.js</a>.</h5>

## Features
* vsc-mock allows you to generate data for:
  * basic values (numbers/letters/vvalues)
  * finance
  * location
  * misc
  * mobile
  * person
  * text
  * things (animals)
  * time
  * web
